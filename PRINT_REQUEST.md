# Suggested CS3 Print Request

All files are in the `print/` folder and are ready to print.

## Recommended quantities

| File | Size | Color | Quantity | Use |
|---|---:|---|---:|---|
| `poster-letter.pdf` | US Letter | Color | 3 | Plastic sign-holders and table display |
| `interaction-board-24x36.pdf` | 24 × 36 in | Color | 1 | Dot-voting and sticky-note board |
| `kids-coloring-page.pdf` | US Letter | B&W or color | 25–40 | Activity for younger children |
| `street-transportation-explorer.pdf` | US Letter | Color | 10–15 | Street-safety learning and drawing |
| `kid-prompt-cards.pdf` | US Letter | Color | 3–4 | Cut into facilitator prompt cards |
| `route-hero-badge-sheet.pdf` | US Letter | Color | 5 | Optional take-home cards |
| `caregiver-survey.pdf` | US Letter, 2-sided if possible | B&W or color | 15–20 sets | Paper backup |

## Setup notes

- Put posters in plastic sign-holders because of wind.
- Secure the 24 × 36 interaction board to a big bulletin pad.
- Use dot stickers in the six response areas.
- Use sticky notes only in the yellow optional-response area.
- Keep paper caregiver surveys separate from any optional pilot-contact information.
