# 0613kidfest_RECOVERY

Recovered Kidova NYC KidsFest website and event materials.

## Website

`index.html` is the QR-code landing page. It includes:

- Safe Route Buddy child activity with retry-until-safe logic
- Anonymous caregiver survey with Google Apps Script submission
- Published Figma prototype embedded at the top of Explore
- Two updated event posters
- Route Hero badge and coloring activity

## Recovery Sources

This version was rebuilt from the GitHub Desktop auto-stash, browser-cached HTML,
and the latest manually exported poster, badge, and Figma embed files.

## Privacy

Do not collect a child's name, school, home address, exact route, or live
location. Do not commit exported response data or pilot contact data.
