# Kidova Interactive Boards: On-Site Use

Print all three boards in color, landscape, US Letter, at 100% scale. Place each Letter guide at the top of a larger bulletin pad or foam board.

## Board 1: School Trip Challenges

- Place the Letter guide above an open sticky-note area.
- Supplies: large sticky notes and thick markers.
- Ask each family to write or draw one challenge.
- Suggested prompt: "My trip would be easier or safer if..."

## Board 2: Draw Your Route Hero

- Use the Letter sheet itself for drawing, or enlarge it if more drawing space is available.
- Supplies: washable markers.
- Invite each child to choose one circle and draw a Route Hero.
- Do not ask children to add real names.

## Board 3: Parent Independence Dot Vote

- Place the Letter guide on a rigid surface.
- Supplies: small dot stickers in one color.
- Ask each caregiver to place one dot in each row, for three dots total.
- Keep the questions anonymous and avoid collecting identifying route details.

## Recommended Table Order

1. Children begin with the Route Hero drawing activity.
2. Caregivers complete the three-row dot vote.
3. Families add an optional sticky note about a school-trip challenge.
4. Invite caregivers to scan the survey QR code for more detailed feedback.
